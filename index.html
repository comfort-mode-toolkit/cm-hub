Atomic Utility Classes

These are the individual building blocks that can be applied to any element:

.cm-font-size-large
This text uses large font size
.cm-font-family-hyperlegible
This text uses Atkinson Hyperlegible font
.cm-color-scheme-dark
This uses dark color scheme
.cm-font-size-xl
Extra large text example
.cm-font-family-serif
Serif font family example
.cm-color-scheme-high-contrast
High contrast color scheme

Composed Utility Classes

These classes combine multiple atomic utilities for common use cases:

.cm-comfort
Base comfort mode with improved line height and letter spacing
.cm-text.cm-text-large
Comfortable text with large size
.cm-text.cm-text-hyperlegible
Comfortable text with accessible font
.cm-all
This applies all comfort settings at once including:
  • Large font size
  • Hyperlegible font family
  • Improved line height
  • Better letter spacing

Technical Information

Plugin Architecture

  • Modular: Each feature is a separate plugin
  • Configurable: Settings controlled via cm-config.json
  • DRY SCSS: Atomic classes reused in composed classes
  • Extensible: Easy to add new plugins

Current Plugins

  • font-size.js: Text size adjustments
  • font-family.js: Accessible font families
  • color-scheme.js: Light/dark/high-contrast themes

Keyboard Shortcuts

  • Alt + C: Toggle Comfort Mode
  • Escape: Close control panel
  • Tab: Navigate controls
